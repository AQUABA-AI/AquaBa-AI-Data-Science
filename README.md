# AquaBa – Data Science & AI Module


## Overview

This folder contains the complete **AI core** for AquaBa:
- **Demand Forecasting** using LSTM (predicts future fish demand)
- **Spoilage Alert System** using rule-based logic + Isolation Forest anomaly detection
- Data preprocessing and synthetic data generation

These modules are designed to be called by the **Backend** and **DevOps** teams via Python imports or API endpoints.
